# CV PDF Required

**IMPORTANT**: This directory should contain the CV PDF file.

## Action Required

Please add the file `Muhammad_Munsif_s_CV.pdf` to this directory (`/assets/cv/`).

The CV page at `/cv/` and the home page both link to `/assets/cv/Muhammad_Munsif_s_CV.pdf` for download.
Without this file, the download buttons will result in a 404 error.

## Expected File

- **Filename**: `Muhammad_Munsif_s_CV.pdf`
- **Location**: `/assets/cv/Muhammad_Munsif_s_CV.pdf`
